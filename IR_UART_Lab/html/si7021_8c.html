float si7021_temperatureF ( void  )

Converts read data to temperature in.

This function will convert the private si7021_i2c_data variable passed into i2c_start function into a fahrenheit value. It uses the algorithm mentioned in the si7021's data sheet to convert into celsius first. That value is then converted into fahrenheit.
