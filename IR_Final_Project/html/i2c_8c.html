Starts i2c communication.

This function will begin i2c serial communication between two devices. It supports both reading and writing

Note
This functions will only function while the i2c peripheral is in the IDLE state
Parameters
[in]i2cThis will be the i2c peripheral being used (I2C0 or I2C1)
[in]device_addressaddress of the slave device
[in]register_addressaddress for register to write to or command for read operation
[in]rwboolean value for read or write operation
[in]rx_datapointer to store data from read operation
[in]tx_datapointer to store data for write operation
[in]num_byteslength of bytes for operation
[in]event_cbevent callback to add to scheduled events once i2c operation comes to an end
