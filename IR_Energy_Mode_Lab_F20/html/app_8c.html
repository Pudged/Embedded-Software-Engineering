static

Sets up a PWM signal on the LETIMER.

This function sets up a struct of APP_LETIMER_PWM_TypeDef that is then used to setup up the specific LETIMER to be used.

Note
This function is called in the peripheral setup function
Parameters
[in]periodPWM period time
[in]act_periodPWM active period time
[in]out0_routeThe routing location for PF5
[in]out1_routeThe routing location for PF4
